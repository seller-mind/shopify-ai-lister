### 第一步：下载素材

### 第二步：进入Partners Dashboard
1. 打开 https://partners.shopify.com 登录
2. 点击 **Apps** → 点击 **WISMO AI**（或 Haimo AI Lister）
3. 左侧菜单点击 **Distribution**
4. 点击 **Manage listing**（或 Manage submission）

### 第三步：填写Listing内容

#### 1) App Icon
- 上传 `app_icon_1200.png`（1200×1200）

#### 2) Desktop Screenshots（3-6张，1600×900）
按顺序上传：
1. `screenshot_dashboard.png` — Alt text: "WISMO AI Dashboard showing conversation analytics and auto-resolved rate"
2. `screenshot_settings.png` — Alt text: "Settings page with 20+ language options and 8 brand color presets"
3. `screenshot_plans.png` — Alt text: "Flexible pricing plans starting from $15/month with free trials"

#### 3) App Card Subtitle（80字符内）
```
AI chatbot that auto-replies "Where is my order?" — instant, multilingual, beautiful.
```

#### 4) Short Description（80字符内）

Automate 50% of support tickets. AI order tracking in 20+ languages.

#### 12) Test Instructions（审核人员用）
```
1. Install the app on your test store
2. Go to Online Store → Themes → Customize
3. Enable the "WISMO AI Chat Widget" App Embed in the theme editor
4. Visit the storefront — click the chat bubble in the bottom-right corner
5. Enter any order number to see the AI order tracking response
6. In Shopify Admin, go to Apps → WISMO AI to see the Dashboard with conversation analytics
7. Check Settings page to configure languages, colors, and return policy URL
```
